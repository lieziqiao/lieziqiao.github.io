给过去的2020做个总结吧

2021-02-06- 2021-02-07
-
随笔

2020 已经过去, 农历新年也即将到来了, 给过去的2020 做个总结吧, 看看自己成长了什么.

2019给自己立下的目标

当时给自己定下的目标是成为一名独当一面的测试工程师, 也确确实实做到了. 能够一个人独自负责一个项目, 让项目平平稳稳度过.
我那个项目, 产品经理都没了, 客户给出来的需求只是个粗糙的需求, 自己把需求给具体完善了, 还要看看是否和原有的一些业务逻辑是否有冲突.
以前别人说我项目掌控力不足, 现在项目掌控力已经比以前强了很多了.

认识到自己的不足

随着自己后面学习的深入, 越发的发现自己的不足, 自己的能力还是差很多. 花时间学了一下数据结构, 重新深入学习一下python 的底层运行机制. 越是学习, 越是知道自己的不足. 路漫漫其修远兮, 吾将上下而求索.

展望2021

给自己的2021 定个目标, 学完 PyQt5 + QML , 之后就深入学习一下数据解析. 最近在做一个旧的项目, 是webservices 的一个项目, 里面的xml 数据不是标准的xml,不是那种像 html 类型的 xml, 我要造数据, 一时间无法下手. 百度了很多方法,对于这种类型的xml ,只能读取之后, 重新生成一个新的xml.
我当时想了一晚上, 百度了很多方法没有搞定. 然后想到了, 浏览器可以解析 xml 数据, 就借助浏览器解析后, 用 selenium 爬取回来. 将爬取回来的数据正则提取回来, 然后将提取的数据用 str 的 api 替换了想要 替换的数据, 将这个 替换后的数据 write 进文件里, 重复多次追加, 批量数据就造出来了.
这些都是投机取巧的方法, 关键还是需要深入学习一下数据解析.

给自己打个鸡血吧

临渊羡鱼,不如退而结网.在想到和得到之间,永远差一个做到, 不努力就会裹布不前.想要成为一名优秀的测试开发工程师, 自己要学习的还有很多, 还需努力.

各位都是白嫖党, 打赏就免了
